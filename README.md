**Setup:**

```sh
$ make install
$ npm link
```

**Play:**

```sh
$ brain-even
```
```sh
$ brain-calc
```
```sh
$ brain-gcd
```
```sh
$ brain-prime
```
```sh
$ brain-progression
```

**Setup:**

```sh
$ make install
$ npm link
```

**Play:**

```sh
$ make brain-even
```
```sh
$ make brain-calc
```
```sh
$ make brain-gcd
```
```sh
$ make brain-prime
```
```sh
$ make brain-progression
```
